# Cyberpunk Edgerunners Moon - Lively Wallpaper

A souvenir for those who have gone through the journey together. 

"俺が君を月に、連れて行くよ！約束する！"

`"I'll take you to the moon! I promise!"`

## Preview
![Preview](preview.gif)

## Features
- **3840x2160@15fps (17211kbps)**
- **Visual Glitch Effect**
- **Customizable Clock**: Configurable position, 12/24 hour formats, and date display.
- **Customizable Caption**: Configurable text, text direction (vertical/horizontal), and screen position.
- **Background Music**: Includes background music with adjustable volume.

## Installation
1. Install [Lively Wallpaper](https://rocksdanister.github.io/lively/).
2. Drag and drop the downloaded `.zip` or this folder into the Lively Wallpaper application.
3. Apply and customize via the Lively control panel.
